# Список звонков сотрудника за выбранный период

Приложение **employers_calls** позволяет вывести на страницу список звонков определенного сотрудника из вашего Битрикс 24.
Отличается от стандартного функционала системы тем, что выводит в строке с информацией о звонке данные о компании, с которой был совершен звонок.
При обращении к скрипту **employers_calls/index.php** отображается форма со списком сотрудников, а также поля для установки границ временного периода.
При отправке формы идет обращение к вебхуку и формируется таблица с данными об исходящих и входящих звонках выбранного сотрудника за определенный период. Все работает без перезагрузки страницы. Над таблицей размещена кнопка, которая позволяет выгрузить полученные данные в файл Excel. Для этого используются библиотеки <a href="https://github.com/aVadim483/fast-excel-writer">avadim/fast-excel-writer</a> и <a href="https://github.com/aVadim483/fast-excel-reader">avadim/fast-excel-reader</a>.

Перечень сотрудников в выпадающем списке формы формируется **динамически**.
Для его настройки вы можете открыть скрипт **employers_calls/admin_employers.php**. В нем выбрать отдел из структуры компании, размещенного в левом столбце и далее выбрать сотрудника из выбранного отдела в правом столбце.
Вы можете выбирать сотрудников из разных отделов, каждый раз сохраняя свой выбор.

### Использование
Создайте входящий вебхук в вашем Битрикс24. И предоставьте доступ к следующим пунктам:
- CRM (crm)
- Телефония (telephony)
- Пользователи (user)
- Структура компании (department)
Сохрание и скопируйте путь к вебхуку.

Создайте в корне вашего сайта папку **/employers_calls/**. Разместите файлы из репозитория в нее.
Откройте файл **/employers_calls/data/settings.php** и укажите в значении константы **C_REST_WEB_HOOK_URL** путь к входящему вебхуку вашего Битрикс24, который вы создали раньше.
Также вы можете указать идентификатор сотрудника, который будет показан в выпающем списке по умолчанию. Для этого заполните константу **DEFAULT_EMPLOYER_ID**.
